== Description ==

WooCommerce Extension Plugin - Add an option to product to be "call for quote" product, and disable online ordering

= Notes =
*If you are theme is already overwriting the following woocommerce template, you need to reoverwrite the templates using the file
located under this plugin's woocommerce folder.
